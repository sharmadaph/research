# Gomoku Replay Demo

Simple demo implementing automatic recording and replay of Gomoku moves using React + MobX.

Features:

- Auto-record moves while a game is in progress
- After the game finishes, recorded moves are available for replay
- Controls: Play (`.replay-play-btn`), Pause (`.replay-pause-btn`), Slider (`.replay-slider`), Current move display (`.current-move`)
- Replay interval: 1000 ms between moves

Run:

1. npm install
2. npm run dev
