Todo Rules

Add tasks and optionally attach rules that auto-mark them complete.

    Rule types

    • Contains: marks complete when task text includes the value.
    • Regex: provide a JS-style regex (no flags). Example: ^Buy
    • Words > N: provide a number N.
    • Expression: a safe boolean expression using variables words, length, digits. Example: words > 3 && length <= 120.
    Tasks are stored locally in your browser.
